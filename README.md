## Installation

1- Clone the repository:
git clone https://github.com/Abdalrzakalsouki/BuildingRestAPI

2- Navigate to project directory:
cd BuildingRestAPI

3- Trust the development certificate:
dotnet dev-certs https --trust

4- Run the app:
dotnet watch run
